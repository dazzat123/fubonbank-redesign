## 🎨 設計理念

### Apple 風格元素
1. **極簡導航** - 精簡至主要類別
2. **大標題排版** - 48-56px 英雄標題
3. **大量留白** - 呼吸空間充足
4. **圓角設計** - 最大 28px 圓角
5. **平滑動畫** - 0.3s 過渡時間
6. **毛玻璃效果** - backdrop-filter blur
7. **漸層色彩** - 品牌色漸層
8. **高質量圖片** - 專業攝影

### 品牌色彩
- **富邦橙**: #F47920
- **黑色**: #1D1D1F
- **白色**: #FFFFFF
- **灰色系**: Apple 風格中性色
